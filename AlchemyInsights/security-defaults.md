# <a name="security-defaults"></a>Privzete varnostne nastavitve

Če želite več informacij o varnostnih privzetih nastavitvah, glejte [Kaj so privzete nastavitve varnosti?](https://docs.microsoft.com/azure/active-directory/conditional-access/concept-conditional-access-security-defaults).

Varnostne privzete nastavitve v imeniku Azure Active Directory (Azure AD) Poenostavite varnost in zaščitite svojo organizacijo. Varnostne privzete vrednosti vsebujejo vnaprej konfigurirane varnostne nastavitve za pogoste napade.

Microsoft pripravlja varnostne privzete nastavitve za vse. Cilj je zagotoviti, da imajo vse organizacije osnovno raven varnosti omogočeno brez dodatnih stroškov. Vklopite varnostne privzete nastavitve v portalu Azure.
