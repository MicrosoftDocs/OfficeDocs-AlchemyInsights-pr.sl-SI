# <a name="app-configuration-policies-for-microsoft-intune"></a>Pravilniki za konfiguracijo aplikacije za Microsoft Intune

Pravilniki za konfiguracijo aplikacij v storitvi Microsoft Intune ponujajo nastavitve konfiguracije za aplikacije s sistemom iOS ali Android. Te nastavitve konfiguracije omogočajo prilagajanje aplikacije. Podpora pravilnika za konfiguracijo je omejena z različicami operacijskega sistema in aplikacijami.

Če želite več podrobnosti, [glejte Pravilniki za konfiguracijo aplikacije za Microsoft Intune](https://docs.microsoft.com/intune/app-configuration-policies-overview).
