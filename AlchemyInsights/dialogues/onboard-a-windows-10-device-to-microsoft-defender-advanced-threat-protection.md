# <a name="onboard-a-windows-10-device-to-microsoft-defender-advanced-threat-protection-mdatp"></a>V napravo s sistemom Windows 10 napredna zaščita pred grožnjami Microsoft Defender (MDATP)

MDATP podpira ta orodja za uvajanje in metode za Windows 10:

- Pravilnik skupine
- Microsoft Endpoint Configuration Manager
- Upravljanje mobilnih naprav, vključno z Microsoft InTune
- Lokalni skripti

Če želite izvedeti več, glejte [Orodja in načini za naprave s sistemom Windows 10](https://go.microsoft.com/fwlink/?linkid=2143460).
