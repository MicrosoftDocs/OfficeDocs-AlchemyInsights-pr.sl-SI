# <a name="enable-or-disable-forms-pro"></a>Omogočanje ali onemogočanje obrazcev Pro

Če poskušate omogočiti Microsoft Forms Pro, je vključen kot del licence za [podjetje Dynamics 365 Enterprise](https://docs.microsoft.com/forms-pro/purchase#purchase-microsoft-forms-pro-for-users-in-a-dynamics-365-tenant) . Alternativno lahko skrbnik kupi [licenco za Microsoft Forms Pro](https://docs.microsoft.com/forms-pro/purchase#purchase-microsoft-forms-pro-for-users-in-a-dynamics-365-tenant). [Licenco za Microsoft Forms Pro lahko odstranite](https://docs.microsoft.com/forms-pro/purchase#disable-microsoft-forms-pro-for-a-user-1) tudi od uporabnika.
