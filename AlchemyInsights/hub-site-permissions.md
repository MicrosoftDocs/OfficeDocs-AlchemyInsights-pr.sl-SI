# <a name="hub-site-permissions"></a>Dovoljenja mesta zvezdišča

Upravljanje dovoljenj spletnega mesta zvezdišča je odvisno od osnovne vrste mesta. Če je spletno mesto skupine, ki je povezano s skupino, morate upravljati dovoljenja prek skupine Microsoft 365. Če gre za spletno mesto za komunikacijo, morate upravljati dovoljenja prek SharePointovih skupin.

- [Dovoljenja mesta zvezdišča](https://docs.microsoft.com/sharepoint/modern-experience-sharing-permissions#hub-site-permissions)  
- [Načrtovanje mest zvezdišča](https://docs.microsoft.com/sharepoint/planning-hub-sites)
