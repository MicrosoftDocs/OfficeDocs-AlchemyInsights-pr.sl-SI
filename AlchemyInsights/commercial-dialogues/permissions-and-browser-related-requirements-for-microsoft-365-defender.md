# <a name="permissions-and-browser-related-requirements-for-microsoft-365-defender"></a>Dovoljenja in zahteve, povezane z brskalnikom za Microsoft 365 Defender

- Dovoljenja: Če želite vklopiti Microsoft 365 Defender, morate biti globalni skrbnik ali varnostni skrbnik v storitvi Azure Active Directory. Če si želite ogledati seznam vlog, ki lahko uporabljajo Microsoft 365 Defender, in če želite izvedeti, kako je dostop do podatkov urejen, glejte [upravljanje dostopa do storitve Microsoft 365 Defender](https://go.microsoft.com/fwlink/?linkid=2143626).

- Zahteve, povezane z brskalnikom: Če želite dostopati do Microsoft 365 Defender v Varnostnem središču za Microsoft 365, uporabite Microsoft Edge, Internet Explorer 11 ali kateri koli HTML 5 – združljiv spletni brskalnik.
