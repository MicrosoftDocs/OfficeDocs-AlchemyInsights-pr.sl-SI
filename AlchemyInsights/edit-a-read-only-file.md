# <a name="edit-a-read-only-file"></a>Urejanje datoteke samo za branje

V knjižnicah dokumentov, ko kliknete ime dokumenta, ga lahko Officeov program nepričakovano odpre kot datoteko samo za branje. Če ga želite odpreti za urejanje, kliknite **Odpri** (tri pike) in nato kliknite **Uredi.**
