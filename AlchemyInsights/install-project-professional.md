# <a name="install-project-professional"></a>Namestitev projekta Professional

Glejte [Nameščanje projekta](https://support.office.com/article/install-project-7059249b-d9fe-4d61-ab96-5c5bf435f281) , če želite izvedeti, kako namestite naročniško različico Project Online, in različico, ki ni naročnina projekta 2019, 2016 ali 2013. 

Za Project 2010 ali 2007 sledite korakom v [Officeu 2010](https://support.office.com/article/install-office-2010-1b8f3c9b-bdd2-4a4f-8c88-aa756546529d) ali [Office 2007](https://support.office.com/article/install-office-2007-88a8e329-3335-4f82-abb2-ecea3e319657). 

Projekt se prodaja kot samostojna aplikacija, tako da če ga še nimate, [kupite ali poskusite Project](https://products.office.com/project).
