# <a name="refresh-using-web-connector-doesnt-work-properly"></a>Osvežite z uporabo Web connector ne deluje pravilno

Če imate priključek spletni skript, ki uporablja funkcijo [Web.Page](https://msdn.microsoft.com/library/mt260924.aspx) in posodobite nabor podatkov ali poročilo po 18. novembra 2016, morate uporabiti prehod v vrstnem redu za osvežitev za pravilno delovanje.

Za več informacij:[https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios](https://docs.microsoft.com/power-bi/refresh-troubleshooting-refresh-scenarios)
