# <a name="use-sara-to-diagnose-and-resolve-outlook-authentication-issues"></a>Uporabite SaRA diagnosticirati in odpraviti težave s preverjanjem pristnosti Outlook

Priporočamo, da uporabljate [Outlook ohranja prosi za geslo](https://aka.ms/SaRA-OutlookPwdPrompt) diagnostičnih prizadetih stroj za odpravljanje težav kjer Outlook nenehno sprašuje po geslu. Ta [SaRA](https://diagnostics.office.com/#/) diagnostične ali samodejnimi in vrne možne rešitve za uporabo za odpravo ugotovljenih težav.
