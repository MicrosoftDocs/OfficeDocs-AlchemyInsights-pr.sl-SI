# <a name="make-microsoft-bing-your-default-search-engine"></a>Nastavitev storitve Microsoft Bing za privzeti iskalnik

Če želite, da je Bing vaš privzeti iskalnik za Microsoft Edge, Google Chrome ali Internet Explorer, upoštevajte navodila v članku Nastavitev brskalnika [Bing za privzeti iskalnik](https://go.microsoft.com/fwlink/?linkid=2148834).
