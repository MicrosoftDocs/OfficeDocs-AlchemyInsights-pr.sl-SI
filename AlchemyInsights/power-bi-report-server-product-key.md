# <a name="power-bi-report-server-product-key"></a>Ključ izdelka za Power BI Report Server

Če ste kupili Power BI  Premium, imate dostop do ključa izdelka Power BI Report Server na zavihku Nastavitve Premium na skrbniškem portalu za Power BI. To je na voljo le za globalne skrbnike ali uporabnike z vlogo skrbnika storitve Power BI.

Če **izberete ključ strežnika Power BI Report Server,** se prikaže pogovorno okno s ključem izdelka. Lahko jo kopirate in uporabite z namestitvijo.

Če imate pogodbo za SQL Server Enterprise SA, lahko ključ izdelka pridobite iz storitvenega središča [za količinsko licenciranje.](https://www.microsoft.com/Licensing/servicecenter/)

Če želite več informacij: https://docs.microsoft.com/power-bi/report-server/install-report-server
