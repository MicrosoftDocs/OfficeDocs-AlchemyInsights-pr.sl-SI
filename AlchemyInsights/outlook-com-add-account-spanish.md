# <a name="agregar-sus-otras-cuentas-de-correo-electrnico"></a>Agregar sus otras de cuentas correo ElectrĂłnico

Cuando agregue otras cuentas de correo ElectrĂłnico Outlook.com, ali lahko usar verižna srajca y Outlook.com para leer mensajes de correo ElectrĂłnico de estas cuentas greh mestu salir de Outlook.com. Cada cuenta que se agrega Outlook.com se denomina una cuenta conectada. También ali lahko [crear ZN alias de correo ElectrĂłnico](https://support.office.com/es-es/article/agregar-o-quitar-un-alias-de-correo-electrónico-en-outlook-com-459b1989-356d-40fa-a689-8f285b13f1f2?wt.mc_id=Office_Outlook_com_Alchemy) que utiliza la misma Bandeja de entrada, verižna srajca, elementos y lista de contactos como su cuenta de Outlook.com existente.

[Lea cómo agregar sus otras cuentas de correo ElectrĂłnico a Outlook.com.](https://support.office.com/es-es/article/agregar-sus-otras-cuentas-de-correo-electrónico-a-outlook-com-c5224df4-5885-4e79-91ba-523aa743f0ba?ui=es-ES&rs=es-ES&ad=ES?wt.mc_id=Office_Outlook_com_Alchemy)
