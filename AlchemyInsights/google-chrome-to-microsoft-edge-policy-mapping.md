# <a name="learn-about-mapping-between-google-chrome-policies-and-microsoft-edge-policies"></a>Več informacij o preslikavi med pravilniki Google Chroma in pravilniki o prednostih Microsoft Edge

Pravilniki za Google Chrome so preslikani v ustrezne Microsoftove pravilnike Edge, ki so podprti v različici 80 brskalnika. Če želite izvedeti več, glejte [Google Chrome na Microsoftovo preslikavo pravilnika Edge](https://go.microsoft.com/fwlink/?linkid=2141933).
