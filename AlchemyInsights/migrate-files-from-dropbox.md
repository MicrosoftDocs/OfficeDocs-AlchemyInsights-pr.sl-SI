# <a name="migrate-files-from-dropbox"></a>Selitev datotek iz Dropboxa

Če selite iz Dropbox v SharePoint online, lahko uporabite funkcijo **Mover**.

Če želite več informacij, glejte [avtorizacija priključka Dropbox](https://docs.microsoft.com/sharepointmigration/mover-dropbox).
