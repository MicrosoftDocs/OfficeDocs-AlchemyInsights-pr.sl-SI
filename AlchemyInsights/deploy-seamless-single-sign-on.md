# <a name="enable-seamless-sso"></a>Omogočiti nemoteno SSO

Sinje Active Directory (Azure AD) brezšivne enotne prijave (brezšivne SSO) samodejno prijavi v uporabnik čas oni so na mizah podjetij, ki so povezani z omrežjem podjetja.
  
[Sledite korakom v tem članku omogočiti nemoteno SSO](https://docs.microsoft.com/azure/active-directory/connect/active-directory-aadconnect-sso-quick-start)
