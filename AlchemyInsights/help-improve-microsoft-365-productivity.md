# <a name="help-improve-microsoft-365-productivity"></a>Pomoč pri izboljšanju produktivnosti Microsoft 365

Ocena produktivnosti podpira digitalno preoblikovanje vaše organizacije z vpogledi v uporabo izdelkov Microsoft 365 in izkušenj tehnologije za njimi.

Več informacij nadete v tem članku:

- [način izračuna rezultata](https://docs.microsoft.com/microsoft-365/admin/productivity/productivity-score)
- [kako razložiti oceno svoje organizacije](https://docs.microsoft.com/microsoft-365/admin/productivity/productivity-score)
- [Kontrolniki zasebnosti za oceno produktivnosti](https://docs.microsoft.com/microsoft-365/admin/productivity/privacy)

Če želite več informacij, glejte [pregled rezultata Microsoftove produktivnosti](https://docs.microsoft.com/microsoft-365/admin/productivity/productivity-score).
