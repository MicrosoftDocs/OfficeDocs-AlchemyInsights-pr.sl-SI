# <a name="copy-and-paste-in-outlookcom"></a>Kopiraj in prilepi v Outlook.com

V Outlook.com ne kopirate ali prilepite gumbov, lahko pa uporabite bližnjice na tipkovnici:

**Na osebnem računalniku:** CTRL + C kopirati, CTRL + X za rezanje, in CTRL + V prilepiti.

**Na Mac:** CMD + C kopirati, CMD + X za zmanjšanje, in CMD + V prilepiti.
