# <a name="multiple-active-sessions-to-the-same-mailbox"></a>Več aktivnih sej v isti nabiralnik

Za nadzor uporabe Exchangeevih virov ima nabiralnik »predračun«.

Izjema pred proračunskim proračunom se lahko sproži zaradi naslednjih okoliščin, vendar ni omejena na:

- V isti seji programa Outlook Web App je odprtih nekaj zavihkov brskalnika.

- Nekaj aktivnih sej programa Outlook Web App v isti nabiralnik.

- Nabiralnik hkrati dostopa do še nekaj drugih odjemalskih aplikacij (Outlook, Outlook Mobile, odjemalski program drugega ponudnika).

- Dolge operacije, kot je izvajanje zahtev za iskanje, se izvajajo iz druge aktivne seje nabiralnika.
