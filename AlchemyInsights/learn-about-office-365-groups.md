# <a name="learn-about-microsoft-365-groups"></a>Več o Microsoftovih 365 skupinah

Microsoft 365 groups je storitev, ki deluje z Officeovimi orodji 365, ki jih uporabljate že, tako da lahko sodelujete s svojimi soigralci pri pisanju dokumentov, ustvarjanju preglednic, delu na projektnih načrtih, sestankih razporejanja ali pošiljanju e-pošte. Če želite več informacij, glejte [več o microsoftovih 365 skupinah](https://support.office.com/article/b565caa1-5c40-40ef-9915-60fdb2d97fa2).
