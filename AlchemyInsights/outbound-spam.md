# <a name="outbound-spam"></a>Odhodna neželena pošta

Če vidite lažne pozitivne izraze za odhodno pošto (dobra odhodna sporočila, ki so označena kot neželena pošta), odprite vstopnico za podporo z vzorčnimi sporočili.
