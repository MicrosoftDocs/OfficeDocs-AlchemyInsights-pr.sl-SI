# <a name="outbound-spam"></a>Odhodna vsiljena pošta

Če vidite lažne pozitivne za odhodno neželeno pošto (dobra odhodna sporočila so označena kot vsiljena pošta), odprite vstopnico za podporo z vzorčnimi sporočili.
