# <a name="install-onedrive"></a>Namestite OneDrive

Odjemalec za sinhronizacijo OneDrive je nameščen z Windows 10. Namestite OneDrive za Windows 7 ali Mac, ali zadobiti prenosen app za vaš telefon ali tablet, pojdite na [http://onedrive.com/download](http://onedrive.com/download).
