# <a name="microsoft-graph-common-issues-and-resolutions"></a>Pogoste težave in resolucije za Microsoft Graph

Če želite dokončati seznam znanih težav z aplikacijo Microsoft Graph, si oglejte [znane težave z aplikacijo Microsoft Graph](https://docs.microsoft.com/graph/known-issues). Če želite druga vprašanja in vire, glejte glavno stran [dokumentacije za Microsoft Graph](https://docs.microsoft.com/graph/) .

- [Priročnik za Microsoft GRAPH REST API v 1.0](https://docs.microsoft.com/graph/api/overview?toc=.%2Fref%2Ftoc.json&view=graph-rest-1.0).
- Preberite, kaj se je spremenilo v API-ju Microsoft Graph iz [Changelog](https://docs.microsoft.com/graph/changelog). 
- [Zastavite vprašanje ali posredujte povratne informacije o API-jih in dokumentih](https://aka.ms/GraphDeveloperSupport) s podporo za razvijalce.

Enotna podpora in stranke Premier lahko zbirajo vstopnico za podporo s tem portalom. Drugi lahko dostopajo do brezplačne podpore za razvijalce za Microsoft Graph na [spletnem mestu Microsoft Q&A](https://aka.ms/AskGraph).
