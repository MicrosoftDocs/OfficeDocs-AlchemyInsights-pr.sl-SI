# <a name="error-onedrive-cannot-connect-to-windows"></a>Napaka: OneDrive ne more vzpostaviti povezave z operacijskim sistemom Windows

Če želite odpraviti sporočilo o napaki, poiščite datoteko cldflt. sys, da preverite, ali se izvaja, in če ni, jo omogočite. 
  
Če želite dodatne informacije o tem, kako omogočiti datoteko cldflt. sys, glejte [onedrive ne more vzpostaviti povezave z operacijskim sistemom Windows» napaka pri dostopu do datotek v storitvi OneDrive.](https://go.microsoft.com/fwlink/?Linkid=2031032)
