# <a name="delete-a-sharepoint-site-collection"></a>Izbrišite zbirko mesta SharePoint

1. V klasičnih SharePoint admin center, na strani zbirke mesta, izberite potrditveno polje ob zbirki mest.
    
2. Na kartici zbirke mest, izberite **Izbriši**.
