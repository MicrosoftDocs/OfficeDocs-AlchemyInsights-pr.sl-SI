# <a name="use-microsoft-intune-to-deploy-microsoft-edge-to-a-macos-device"></a>Uporaba Microsoft InTune za uvajanje Microsoftovega roba v napravo macOS

Če želite uvesti, konfigurirati, nadzorovati in zaščititi program v napravi macOS, morate dodati program v Microsoft InTune. Aplikacija mora biti v obliki, ki jo podpira funkcija InTune, možnosti upravljanja, ki so na voljo v programu InTune, pa so odvisne od vrste aplikacije. Uporaba funkcije intuned pomeni, da vam ne bo treba uporabljati orodja za oblivanje z aplikacijami macOS.

InTune podpira Microsoft Edge. Če želite izvedeti več, glejte [Dodajanje Microsoft Edge v naprave MacOS s programom Microsoft InTune](https://go.microsoft.com/fwlink/?linkid=2134949).
