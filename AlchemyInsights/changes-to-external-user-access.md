# <a name="changes-to-external-user-access"></a>Spreminjanje dostopa zunanjega uporabnika

Z začetkom marca 23, 2018 zunanji uporabniki ne bodo več mogli dostopati do vsebine, ki je bila v skupni rabi z vsemi, vsemi preverjenimi uporabniki ali vsemi uporabniki obrazcev, tudi če so bili povabljeni ali dodani pooblaščenim skupinam. Zunanji uporabniki bodo lahko dostopali do vsebine, ki je bila v skupni rabi z uporabnikom ali skupinam, ki jim pripada uporabnik. Če želite ohraniti obstoječe nastavitve, si oglejte: https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
