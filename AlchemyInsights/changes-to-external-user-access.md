# <a name="changes-to-external-user-access"></a>Spremembe zunanjih uporabnikov dostop

Odhod marec 23, 2018, zunanjim uporabnikom ne bo omogočen dostop do vsebine, ki je bila deljena z vsem, vse preverjene uporabnike, ali vse oblike uporabnikov, tudi če so povabljeni ali dodati pooblaščene skupine. Zunanji uporabniki le lahko dostop do vsebine, ki je bila deljena z uporabnika ali skupine, ki ji pripada uporabnik. Če bi želeli obdržati obstoječe nastavitve, si oglejte:https://support.microsoft.com/help/4089534/how-to-grant-the-everyone-claim-to-external-users-in-office-365
