# <a name="create-users-in-hybrid-deployments"></a>Ustvarjanje uporabnikov v hibridnih uvedbah

Ko je vaša organizacija hibridna uvedba med okoljem na mestu uporabe in sistemom Microsoft 365, morate ustvariti vse uporabniške predmete v imeniku Active Directory na mestu uporabe. Predmete uporabnika ustvarite v ustrezni organizacijski enoti (OU), tako da so sinhronizirani v vaši organizaciji.
