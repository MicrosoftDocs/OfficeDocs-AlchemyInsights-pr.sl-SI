# <a name="create-users-in-hybrid-deployments"></a>Ustvarjanje uporabnikov v hibridnih uvajah

Ko je vaša organizacija hibridna razmestitev med krajevnim okoljem in Microsoft 365, morate ustvariti vse uporabniške predmete v krajevnem imeniku Active Directory. Uporabniške predmete ustvarite v ustrezni organizacijski enoti (OU), tako da so sinhronizirani v vašo organizacijo.
