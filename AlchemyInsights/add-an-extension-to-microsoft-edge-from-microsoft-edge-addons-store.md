# <a name="add-an-extension-to-microsoft-edge-from-the-microsoft-edge-add-ons-store"></a>Dodajanje razširitve v Microsoft Edge iz trgovine Microsoft Edge Add-ons

Če želite dodati razširitev na Microsoft Edge iz trgovine Microsoft Edge Add-ons:

- Odprite Microsoft Edge.
- Izberite **... > razširitve > [pridobite pripone za Microsoft Edge](https://go.microsoft.com/fwlink/?linkid=2136408)**.
- Izberite razširitev, ki jo želite dodati, in nato izberite **pridobi**.
- V pogovornem oknu za potrditev previdno preglejte spremembe, ki jih lahko izvede razširitev, in izberite **Dodaj podaljšanje**.

Prejeli boste potrditev, da je bila pripona dodana.
