# <a name="learn-about-site-compatibilityaffecting-changes-coming-to-microsoft-edge"></a>Več informacij o združljivosti mesta – vpliva na spremembe, ki prihajajo na Microsoft Edge

Splet se nenehno razvija, da bi izboljšali uporabniško izkušnjo, varnost in zasebnost. Nekatere spremembe so dovolj pomembne, da vplivajo na funkcionalnost obstoječih strani. Skupina Microsoft Edge posodobi strani, kot jih razvija razmišljanje, časovnice se strdijo in [spremembe so objavljene](https://go.microsoft.com/fwlink/?linkid=2135534).
