# <a name="find-results-in-bing-search-by-using-azure-logic-apps"></a>Iskanje rezultatov v storitvi Bing Search z uporabo aplikacije Azure Logic

S funkcijo Bing Search v programu Logic lahko avtomatizirate obdelavo rezultatov iskanja in omogočite, da so ti rezultati na voljo za druga dejanja. Na primer, lahko poiščete elemente novic in nato Twitter objavite kot Tweets v vašem viru.

Če želite več informacij, glejte Iskanje [rezultatov v storitvi Bing Search z uporabo aplikacij Azure Logic](https://go.microsoft.com/fwlink/?linkid=2151928).
