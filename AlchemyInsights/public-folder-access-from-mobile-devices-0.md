# <a name="public-folder-access-from-mobile-devices"></a>Dostop do javnih map iz mobilnih naprav

Javne mape niso dostopne iz mobilnih naprav.
  
Če želite imeti podobne funkcije kot javne mape in dostopati do nje iz mobilnih naprav, glejte [več o microsoftovih 365 skupinah](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) za boljšo alternativo.
