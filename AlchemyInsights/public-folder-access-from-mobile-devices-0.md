# <a name="public-folder-access-from-mobile-devices"></a>Dostop do javne mape iz prenosnih naprav

Javne mape niso dostopne v prenosnih napravah.
  
Če želite, da so funkcije podobne javnim mapam in jih lahko dostopate iz prenosnih naprav, si oglejte [več informacij o skupinah Microsoft 365](https://support.office.com/article/learn-about-office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2) za boljšo alternativo.
