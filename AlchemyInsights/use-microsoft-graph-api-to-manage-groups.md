# <a name="use-microsoft-graph-api-to-manage-groups"></a>Upravljanje skupin z API-jem za Microsoft Graph

Skupine so zbirke uporabnikov in drugih upravnikov, ki imajo v skupni rabi dostop do virov v Microsoftovih storitvah ali v aplikaciji. Microsoft Graph zagotavlja API-je, ki jih lahko uporabite za ustvarjanje in upravljanje različnih vrst skupin in funkcionalnosti skupine glede na scenarij. Vse operacije, povezane z združevanjem v programu Microsoft Graph zahtevajo soglasje skrbnika. Če želite več informacij, glejte [delo s skupinami v programu Microsoft Graph](https://docs.microsoft.com/graph/api/resources/groups-overview).
