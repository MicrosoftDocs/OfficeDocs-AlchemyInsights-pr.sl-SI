# <a name="proxyaddress-incorrect"></a>ProxyAddress nepravilna

Ko je predmet sinhroniziran s storitvijo Azure AD, se vrednosti, ki so določene v atributu proxyAddresses v imeniku Active Directory, primerjajo s pravili Azure AD in nato je atribut proxyAddresses izpolnjen v storitvi Azure AD. Zato vrednosti atributa proxyAddresses za predmet v imeniku Active Directory morda ne bodo enake vrednosti atributa proxyAddresses v storitvi Azure AD.
  
Če želite izvedeti več o tem, kako je proxyAddress naseljen, si oglejte, [Kako je atribut» proxyAddress «izpolnjen v storitvi AZURE ad](https://support.microsoft.com/help/3190357/how-the-proxyaddresses-attribute-is-populated-in-azure-ad).
