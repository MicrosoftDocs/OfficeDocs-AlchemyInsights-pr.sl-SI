# <a name="workflows-in-sharepoint"></a>Poteke dela v SharePointu

<p>SharePoint poteki dela so pre-programirana mini-aplikacije, ki Poenostavite in avtomatizacijo različnih poslovnih procesov. Poteke dela lahko segajo od zbiranja podpisov, povratne informacije ali soglasij za načrt ali dokument, za sledenje trenutni status rutinski postopek.</p> <p><strong>Uvod:</strong> Če ste&rsquo;re samo uvod v poteke dela SharePoint, si oglejte <a href="https://support.office.com/en-us/article/introduction-to-sharepoint-workflow-07982276-54e8-4e17-8699-5056eff4d9e3">Uvod v SharePoint poteke dela</a> za več informacij.</p> <p><strong>Microsoft pretoka:</strong> Razmislite o uporabi Microsoft Flow namesto klasičnih rešitev SharePoint poteka dela. Za več informacij, glejte <a href="https://support.office.com/en-us/article/create-a-flow-for-a-list-or-library-in-sharepoint-online-or-onedrive-for-business-a9c3e03b-0654-46af-a254-20252e580d01">Ustvarjanje tok za seznam ali knjižnica SharePoint Online ali OneDrive</a>.</p> <p><strong>Potek dela, odpravljanje težav in odpravljanje napak:</strong> Odpravljanje napak poteki dela spominja na prejšnje različice. Več informacij, glejte <a href="https://docs.microsoft.com/en-us/sharepoint/dev/general-development/debugging-sharepoint-server-workflows">Odpravljanje napak SharePoint poteki dela</a>. Več informacij o odpravljanju težav, si oglejte <a title="odpravljanje SharePoint Server napake pri preverjanju veljavnosti poteka dela" href="https://docs.microsoft.com/en-us/sharepoint/dev/general-development/troubleshooting-sharepoint-server-workflow-validation-errors-in-visio">Odpravljanje napak pri preverjanju veljavnosti potek dela strežnika SharePoint Server</a>.&nbsp;</p>
