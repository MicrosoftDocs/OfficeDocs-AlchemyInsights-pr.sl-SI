# <a name="get-help-with-windows-10-desktop"></a>Pomoč za namizno različico sistema Windows 10

- Če želite ustvariti bližnjico na namizju, glejte [Pripenjanje aplikacij in map na namizje ali v opravilno vrstico](https://support.microsoft.com/windows/pin-apps-and-folders-to-the-desktop-or-taskbar-f3c749fb-e298-4cf1-adda-7fd635df6bb0).

- Če želite v sistemu Windows 10 povečati ikone ali besedilo, [glejte Spreminjanje velikosti ikon na namizju.](https://support.microsoft.com/windows/change-the-size-of-your-desktop-icons-85a9d341-2a4f-3d96-c796-ae116a187211)

- Če želite prikazati ikone namizja v sistemu Windows 10, glejte [Prikaz ikon namizja v sistemu Windows 10.](https://support.microsoft.com/windows/show-desktop-icons-in-windows-10-c13270f0-3812-c71d-f27e-29aa32588b20)
