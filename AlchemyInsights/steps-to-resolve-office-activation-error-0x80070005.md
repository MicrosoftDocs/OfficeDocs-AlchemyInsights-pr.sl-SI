# <a name="steps-to-resolve-office-activation-error-0x80070005"></a>Navodila za rešitev napake pri aktivaciji Officea 0x80070005

Če se je napaka 0x80070005 začela po posodobitvi sistema Windows, lahko težavo odpravite tako, da posodobite register.
  
Prenesite in namestite [pomočnika za podporo in obnovitev](https://aka.ms/SARA-OfficeActivation-Alchemy) ter zaženite **orodje za odpravljanje težav z aktiviranjem**.
  
Če želite dodatno pomoč, glejte: [Office error code 0x80070005 pri aktiviranju Officea](https://support.office.com/article/7aa7600f-df57-4aef-81d2-25509c66f865)
