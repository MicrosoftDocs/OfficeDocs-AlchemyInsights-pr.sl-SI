# <a name="troubleshoot-error-code-aadsts50011"></a>Odpravljanje težav s kodo napake AADSTS50011

Če želite razrešiti napako AADSTS50011, izvedite priporočen korak, ki je opisan spodaj.

**AADSTS50011**: InvalidReplyTo – naslov odgovora manjka, je napačno konfiguriran ali pa se ne ujema z naslovi odgovorov, konfiguriranimi za aplikacijo.

Kot ločljivost zagotovite, da dodate ta manjkajoči naslov za odgovor v aplikacijo Azure Active Directory (AD) ali pa imate osebo z dovoljenji za upravljanje aplikacije v programu AD to naredite namesto vas. Če želite več informacij, si oglejte članek Odpravljanje težav z [AADSTS50011](https://docs.microsoft.com/troubleshoot/azure/active-directory/error-code-aadsts50011-reply-url-mismatch).
