# <a name="bulk-delete-user"></a>Množično brisanje uporabnika

Veliko število uporabnikov iz skupine lahko odstranite z datoteko CSV (vrednosti, ločenimi z vejico) za množično brisanje uporabnikov. Če želite izbrisati množične uporabnike s portalom Azure AD, [glejte Množično brisanje uporabnikov v imeniku Azure Active Directory](https://docs.microsoft.com/azure/active-directory/users-groups-roles/users-bulk-delete).
