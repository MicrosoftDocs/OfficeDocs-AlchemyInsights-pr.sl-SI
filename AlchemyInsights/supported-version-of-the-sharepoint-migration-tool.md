# <a name="supported-version-of-the-sharepoint-migration-tool"></a>Podprte različice programa SharePoint migracij orodje

Trenutno orodje za migracijo SharePoint podpira samo iz SharePoint 2013. V tem času ne podpira različice 2010 in 2016.
  
Če želite več informacij, glejte [uvedbo SharePoint migracij orodje](https://go.microsoft.com/fwlink/?linkid=2044765&amp;clcid=0x409).
