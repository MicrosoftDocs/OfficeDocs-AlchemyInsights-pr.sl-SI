# <a name="add-a-sharepoint-calendar"></a>Dodajanje SharePointovega koledarja

SharePointovi koledarji so klasične vrste seznamov, ki so včasih zamenjane s koledarji skupine Microsoft 365.
 
Če želite ustvariti SharePointov koledar, upoštevajte ta navodila:
 
1.  Izberite ikono orodja za **Nastavitve** in nato izberite **Dodaj program**.
2.  V iskalno polje **Najdi program** vnesite **koledar** in pritisnite tipko ENTER.
3.  Izberite program z naslovom **koledar**in nato vnesite ime za svoj koledar.

Če želite več informacij o skupinah Microsoft 365, glejte [več o skupinah microsoft 365](https://support.office.com/article/Learn-about-Office-365-groups-b565caa1-5c40-40ef-9915-60fdb2d97fa2).
