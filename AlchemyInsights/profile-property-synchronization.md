# <a name="profile-property-synchronization"></a>Lastnosti sinhronizacije profila

Ni mogoče prirediti dodatne lastnosti za uvoz oglas za sinhronizacijo iz imenika Active Directory Azure za program uporabniškega profila. Uvoz oglas je omejena na vnaprej niz lastnosti, da se zagotovi dosledno izvajanje posla razporejevalnika.
