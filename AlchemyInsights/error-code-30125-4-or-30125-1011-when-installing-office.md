# <a name="error-code-30125-4-or-30125-1011-when-installing-office"></a>Koda napake 30125-4 ali 30125-1011 namestitvi Officea

Lahko vidite to napako, če protivirusno programsko opremo, požarni zid, proxy nastavitve ali povezavo voditi vi s napeljati urad. Tukaj je nekaj stvari, ki lahko poskušate odpraviti to težavo:
  
- Znova zaženite računalnik in znova namestite Office
    
- Napeljati urad using [offline umestiti urad](https://support.office.com/article/f0a85fe7-118f-41cb-a791-d59cef96ad1c?wt.mc_id=Alchemy_ClientDIA)
    
- Začasno izklopite nastavitve proxyja
    
- Začasno izklopite protivirusno programsko opremo
    
- Začasno izklopite požarni zid
    
Več informacij si oglejte [kodo napake 30125-4 ali 30125-1011 namestitvi Officea](https://support.office.com/article/7bfabec6-76be-4cde-880e-819a9c569612?wt.mc_id=Alchemy_ClientDIA)
