# <a name="error-code-30125-4-or-30125-1011-when-installing-office"></a>Koda napake 30125-4 ali 30125-1011 pri nameščanju Officea

To napako lahko vidite, če vaša protivirusna programska oprema, požarni zid, nastavitve strežnika proxy ali povezava preprečujejo nameščanje Officea. Tukaj je nekaj stvari, ki jih lahko poskusite odpraviti težavo:
  
- Znova zaženite računalnik in znova namestite Office

- Namestitev Officea s programom [Office offline Installer](https://support.office.com/article/f0a85fe7-118f-41cb-a791-d59cef96ad1c?wt.mc_id=Alchemy_ClientDIA)

- Začasno izklopite nastavitve strežnika proxy

- Začasno izklopite protivirusno programsko opremo

- Začasno izklopite požarni zid

Če želite več informacij, glejte [koda napake 30125-4 ali 30125-1011 pri nameščanju Officea](https://support.office.com/article/7bfabec6-76be-4cde-880e-819a9c569612?wt.mc_id=Alchemy_ClientDIA)
