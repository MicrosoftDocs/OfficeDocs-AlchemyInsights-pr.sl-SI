Ko poskusite namestiti Office 365/2016, morda videli to sporočilo o napaki, če naprava ne izpolnjuje [sistemske zahteve](https://products.office.com/office-system-requirements).
  
 **»Nepodprt operacijski sistem«**
  
Office 365/2016 zahteva:
  
PC: Windows 10, Windows 8.1, Windows 8, Windows 7 Service Pack 1, Windows strežnik 2016, Windows Server 2012 R2, Windows Server 2012 ali Windows Server 2008 R2
  
ali
  
Mac OS X 10.10
  
Če med poskusom namestite Office v mobilno napravo ali Chromebook got to vest, glejte [Nastavitev Office aplikacije in e-pošte na mobilni napravi](https://support.office.com/article/7dabb6cb-0046-40b6-81fe-767e0b1f014f?wt.mc_id=Alchemy_ClientDIA.aspx).
