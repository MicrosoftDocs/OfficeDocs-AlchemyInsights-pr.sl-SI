# <a name="solutions-for-unsupported-operating-system"></a>Rešitve za "nepodprt operacijski sistem"

Ko poskušate namestiti Microsoft 365, se lahko prikaže nepodprto sporočilo o napaki **operacijskega sistema** , če naprava ne izpolnjuje [sistemskih zahtev](https://products.office.com/office-system-requirements).
  
Microsoft 365 zahteva:
  
- RAČUNALNIK: Windows 10, Windows 8,1, Windows 8, servisni paket SP1 za Windows 7, Windows Server 2016, Windows Server 2012 R2, Windows Server 2012 ali Windows Server 2008 R2

    Ali

- Mac OS X 10,10

Če imate to sporočilo, medtem ko poskušate namestiti Office v mobilno napravo ali Chromebook, preberite, [kako namestite in zaženete Microsoft Office v Chromebooku](https://support.office.com/article/32f14a23-2c1a-4579-b973-d4b1d78561ad?wt.mc_id=Alchemy_ClientDIA).
