# <a name="help-signing-in-with-windows-hello-pin"></a>Pomoč pri vpisu s kodo PIN za Windows Hello

- Če želite spremeniti, ponastaviti ali obnoviti PIN (ko ste vpisani), obiščite stran Spreminjanje kode PIN, ko ste [že vpisani v napravo.](https://support.microsoft.com/windows/change-your-pin-when-you-re-already-signed-in-to-your-device-0bd2ab85-b0df-c775-7aef-1324f2114b19)

- Če želite spremeniti, ponastaviti ali obnoviti PIN (ob odjavi), obiščite stran Ponastavitev kode PIN, ko niste [vpisani v Windows 10.](https://support.microsoft.com/windows/reset-your-pin-when-you-aren-t-signed-in-to-windows-10-a386c519-3ab2-b873-1e9b-bb228a98b904)

- Če želite nastaviti PIN za Windows Hello, [obiščite spletno mesto »Spoznajte Windows Hello in ga nastavite«.](https://support.microsoft.com/windows/learn-about-windows-hello-and-set-it-up-dae28983-8242-bb2a-d3d1-87c9d265a5f0)
