# <a name="microsoft-viva-connections-availability"></a>Razpoložljivost povezav Microsoft Viva

Povezave Microsoft Viva bodo na voljo za namizje v prvi polovici 2021 in za mobilne naprave poleti 2021. Dodatne funkcije se bodo zagnale tekom 2022.

Če želite več informacij, si oglejte, kako [Microsoft Viva Connections pomaga vsem, da ostanejo zasedeni in obveščeni](https://techcommunity.microsoft.com/t5/microsoft-viva-blog/microsoft-viva-connections-helps-everyone-to-stay-engaged-and/ba-p/2107009).
