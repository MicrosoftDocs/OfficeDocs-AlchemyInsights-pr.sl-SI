# <a name="block-pop-up-windows-in-microsoft-edge"></a>Blokiranje pojavnih oken v brskalniku Microsoft Edge

1. Pojdite na **... (Nastavitve in še več)**  >  **Nastavitve**  >  **Dovoljenja za mesto**.
2. Izberite **pojavnih oken in preusmeritev**.
3. Vklopite **blok**.
