# <a name="how-to-buy-a-copy-of-windows-10"></a>Kako kupiti kopijo Windows 10

Če želite posodobljene cene in navodila za nakup in prenos polne različice Windows 10, obiščite [Microsoft Store](https://www.microsoft.com/store/b/windows).

**Opomba:** Brezplačna Windows 10 s programom Pridobite Windows 10 (GWX) se je končala 29. julija 2016.

**Pomembno:** Če uporabljate Windows 7, Microsoft priporoča, da se premaknete v nov računalnik s sistemom Windows 10, saj je Windows 7 dosegel konec življenjskega cikla podpore 14. januarja 2020.
