# <a name="manage-synchronized-user-accounts"></a>Upravljanje sinhroniziranih uporabniških računov

Po navadi uporabite skrbniško središče za Microsoft 365 za dodajanje in spreminjanje uporabniških računov. Vendar pa ga ne morete uporabiti, če so omogočeni imeniški sychronization ali SSO. Namesto tega upravljajte uporabnike z lupino Windows PowerShell, kot je opisano v [razdelku konfiguracija uporabniških računov v storitvi Office 365 PowerShell](https://docs.microsoft.com/office365/enterprise/powershell/configure-user-account-properties-with-office-365-powershell ).
