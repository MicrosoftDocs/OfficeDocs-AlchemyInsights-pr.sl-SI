# <a name="your-users-mailbox-is-being-created"></a>Vaš uporabnikovega nabiralnika je oživela

To lahko traja eno uro, preden nov nabiralnik je pripravljen za uporabo.
  
Če uporabnikovega nabiralnika je bil "priprava" za več kot eno uro, lahko poskusite brisanje in ponovno ustvarjanje uporabnika ali z možnostjo "naj nas pokličeš" govoriti inženir za podporo.
