# <a name="unable-to-access-the-microsoft-defender-atp-portal"></a>Dostop do portala za Microsoft Defender ATP ni uspel

Če prejmete sporočilo, Niste pooblaščeni za dostop do **portala**, ne pozabite, da je Microsoft Defender ATP nadzor varnosti, raziskovanje dogodkov in odziv, dostop pa je omejen in nadzorovan s strani uporabnika. 

Če želite več informacij, [glejte Dodeljevanje dostopa uporabnika Microsoft Defender Security Center.](/windows/threat-protection/windows-defender-atp/assign-portal-access-windows-defender-advanced-threat-protection)
