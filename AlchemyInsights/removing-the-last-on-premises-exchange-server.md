# <a name="removing-the-last-on-premises-exchange-server"></a>Odstranjevanje zadnji krajevni strežnik Exchange

Preden odstranite zadnji Exchange server iz krajevne organizacije, prepričajte se boste razumeli posledice za to, da bi vam pomagal odločiti, če morate da slednji pomočnik v druge namene. Za več informacij, si oglejte [Kako in kdaj za razgradnjo vaše krajevne strežnike hibridnega uvajanja](https://technet.microsoft.com/library/dn931280%28v=exchg.150%29.aspx).
