# <a name="removing-the-last-on-premises-exchange-server"></a>Odstranjevanje zadnjega strežnika Exchange Server na mestu uporabe

Preden odstranite zadnji Exchangeev strežnik iz organizacije na mestu uporabe, se prepričajte, da razumete posledice tega, da se boste lažje odločili, ali potrebujete ta zadnji strežnik za druge namene. Če želite več informacij, si oglejte, [Kako in kdaj razgradite strežnike Exchange na mestu uporabe v hibridnem uvajanju](https://technet.microsoft.com/library/dn931280%28v=exchg.150%29.aspx).
