# <a name="office-app-icons-or-shortcuts-are-white-blank-or-duplicate"></a>Ikone ali bližnjice aplikacije zbirke Office so bele, prazne ali podvojene

Če so ikone Officeovih programov prikazane prazne ali bele, poskusite [popraviti Officeovo aplikacijo](https://support.office.com/article/repair-an-office-application-7821d4b6-7c1d-4205-aa0e-a6b40c5bb88b). Če želite odstraniti podvojene ikone programa Office, glejte [Officeove bližnjice ostanejo po odstranitvi Officea](https://support.office.com/article/office-shortcuts-remain-after-office-uninstall-cc04b8e2-6e91-4c10-94af-9359e595d565).

Če želite več informacij, glejte [Officeove ikone so prazne po namestitvi Officea iz trgovine Microsoft Store](https://support.office.com/article/office-icons-are-blank-after-installing-office-from-the-microsoft-store-7cdaebde-93d5-4873-b767-d9ddc0474d59).
