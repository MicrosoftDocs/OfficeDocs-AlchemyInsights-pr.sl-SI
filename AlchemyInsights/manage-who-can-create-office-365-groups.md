# <a name="manage-who-can-create-microsoft-365-groups"></a>Upravljanje, kdo lahko ustvari skupine Microsoft 365

Vsi v organizaciji lahko privzeto ustvarijo skupine O365.
  
Če želite to spremeniti, lahko ustvarite varnostno skupino z lupino PowerShell, ki omejuje ustvarjanje skupine v podskupino uporabnikov.
  
Upoštevajte navodila po korakih: [upravljanje, kdo lahko ustvari skupine Microsoft 365](https://docs.microsoft.com/microsoft-365/admin/create-groups/manage-creation-of-groups)
