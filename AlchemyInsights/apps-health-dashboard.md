# <a name="microsoft-365-apps-health-dashboard-in-the-microsoft-365-apps-admin-center"></a>Nadzorna plošča s stanjem aplikacij storitve Microsoft 365 v skrbniškem središču za aplikacije Microsoft 365

Nadzorna plošča s stanjem aplikacij Storitve Microsoft 365 v Skrbniškem središču za Microsoft 365 vam pomaga, da so Officeove aplikacije posodobljene, saj vam omogoča vpogled v njihovo delovanje v odjemalskih napravah in ponuja smernice za optimizacijo in odpravljanje težav. Če želite več informacij, glejte [Pregled skrbniškega središča za Microsoftove aplikacije](https://docs.microsoft.com/deployoffice/admincenter/overview) in stanja aplikacij storitve Microsoft [365.](https://docs.microsoft.com/deployoffice/admincenter/microsoft-365-apps-health)
