# <a name="accessibility-features-in-microsoft-edge"></a>Funkcije pripomočkov za osebe s posebnimi potrebami v Microsoft Edge

Microsoft Edge ima vgrajene možnosti pripomočkov za osebe s posebnimi potrebami, ki vam bodo v pomoč pri brskanju po spletu na način, ki najbolje ustreza vašim potrebam.

Če želite več informacij, glejte [funkcije pripomočkov za osebe s posebnimi potrebami v brskalniku Microsoft Edge](https://go.microsoft.com/fwlink/?linkid=2153648).
