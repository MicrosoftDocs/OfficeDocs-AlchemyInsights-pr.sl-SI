# <a name="get-started-with-microsoft-search"></a>Uvod v Microsoft Search

Z aplikacijo Microsoft Search lahko poiščete osebe, datoteke, organizacijske grafikone, mesta, odgovore na pogosta vprašanja in drugo vsebino, ki vam bo v pomoč pri dokončanju opravil v programu, ki ga uporabljate.

Če želite več informacij, glejte [pregled Microsoftovega iskanja](https://docs.microsoft.com/microsoftsearch/overview-microsoft-search).
