# <a name="download-and-share-updates"></a>Prenos in skupna raba posodobitev

Namesto prenosa posodobitev posamezno, način prenosa optimizacije prenašanja omogoča skupno rabo med več računalniki. Če želite več informacij, [glejte Optimizacija prenašanja za posodobitve sistema Windows 10](https://docs.microsoft.com/windows/deployment/update/waas-delivery-optimization).  

Razpoložljivi načini prenosa vključujejo:  
- Samo HTTP  
- Lan  
- Skupine  
- Internet  
- Preprost  
- Bypass
