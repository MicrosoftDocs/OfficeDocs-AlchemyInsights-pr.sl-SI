# <a name="create-custom-microsoft-surface-operating-system-images"></a>Ustvarjanje slik operacijskega sistema Microsoft Surface po meri

S pospeševalnikom za Microsoft Surface Deployment Accelerator (SDA) lahko poenostavite in avtomatizirate uvajanje novega operacijskega sistema, ki je prilagojen vaši organizaciji in ustreza Microsoftovim priporočilom.

1. V GitHub pojdite na SurfaceDeploymentAccelerator.

1. Izberite **Koda**, nato pa izberite Odpri **z namizjem GitHub ali** Prenesi **ZIP**.

1. Preglejte **README.md,** če želite izvedeti, kako deluje skript.

1. Uredite skript z ustreznimi spremenljivkami za okolje, kot je zabeleženo v datoteki Berime, nato pa ga preglejte, preden ga zaženete v preskusnem okolju.
